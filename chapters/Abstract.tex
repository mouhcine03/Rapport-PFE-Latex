\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This end-of-studies project focuses on the design and implementation of an integration solution between Boomi and Infor ERP in order to automate work order management. The goal is to simplify business processes, reduce errors caused by manual data entry, and improve the overall operational efficiency for the hosting company, \textit{We are beebay}. 

By establishing a reliable synchronization pipeline, this integration guarantees real-time updates and seamless workflow management, empowering stakeholders to efficiently manage orders across departments without human intervention. The implemented solution highlights the significance of iPaaS (Integration Platform as a Service) technologies such as Boomi in modernizing IT enterprise architectures.

\vspace{1.5cm}

\textbf{Keywords:} Boomi, Infor, System Integration, Automation, Work Orders, Data Synchronization, iPaaS.
